âŒ TROUBLESHOOTING

ERROR: "Weights file not found at ..."
CAUSE: Path is incorrect or file doesn't exist
FIX:
  - Check: ls runs/yolov11_wildlife/weights/
  - Ensure you completed at least one training run
  - Use absolute path if relative path fails

ERROR: "Model architecture mismatch"
CAUSE: Incompatible model (wrong YOLO version or different classes)
FIX:
  - Use models trained with same YOLO version
  - Ensure model has 6 classes (Buffalo, Elephant, etc.)
  - Only use Ultralytics YOLO models

ERROR: "CUDA out of memory"
CAUSE: Model + larger batch size exceeds GPU memory
FIX:
  - Reduce batch: --batch 2
  - Reduce image size: --imgsz 1024
  - Clear GPU cache before training
