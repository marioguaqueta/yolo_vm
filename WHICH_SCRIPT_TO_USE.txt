ğŸ’¡ FREQUENTLY ASKED QUESTIONS

Q: Can I switch models mid-training?
A: No. yolo11m and yolo11x have different architectures.
   You must start fresh with yolo11x.

Q: Can I use train_vm_v2.py with yolo11m?
A: Yes! Use: python train_vm_v2.py --model yolo11m.pt --batch 4
   You'll get aerial optimizations with faster training.

Q: Why is train_vm_v2.py slower?
A: yolo11x has 56M params vs yolo11m's 20M (2.8x larger)
   But it gives +6-10% better F1 score!

Q: Can I make train_vm_v2.py faster?
A: Yes! Use smaller image size:
   python train_vm_v2.py --imgsz 1280 --batch 3
   Trade-off: Slightly lower accuracy (-2-3% F1)

Q: Should I always use train_vm_v2.py then?
A: Not always. Use train_vm.py for experiments.
   Use train_vm_v2.py for final production model.

Q: Can I customize the augmentation?
A: Yes! Both scripts support CLI arguments.
   But train_vm_v2.py defaults are already optimized!

Q: What if I have limited GPU memory?
A: Use train_vm.py with yolo11m (needs ~12GB)
   Or use: python train_vm_v2.py --model yolo11l.pt --batch 2
   (yolo11l is between yolo11m and yolo11x)
