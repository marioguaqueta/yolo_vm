================================================================================
 üêõ TROUBLESHOOTING
================================================================================

Problem: Out of Memory (OOM) Error
Solution: Reduce batch size or image size
  $ python train_yolov11_wildlife.py --batch 2 --imgsz 1024

Problem: Training very slow
Solution: Check if GPU is being used
  $ python -c "import torch; print(torch.cuda.is_available())"
  If False, use Google Colab or cloud GPU

Problem: Wandb login fails
Solution: Train without wandb or use API key
  $ python train_yolov11_wildlife.py --no-wandb
  Or:
  $ export WANDB_API_KEY=your_key
  $ python train_yolov11_wildlife.py

Problem: Dataset not found
Solution: Update BASE_DIR in the training script
  Edit train_yolov11_wildlife.py:
    BASE_DIR = Path("/your/actual/path/to/Yolo")

Problem: Missing dependencies
Solution: Install requirements
  $ pip install -r requirements.txt

Problem: Permission denied
Solution: Make scripts executable
  $ chmod +x setup_and_train.sh train_yolov11_wildlife.py test_setup.py
