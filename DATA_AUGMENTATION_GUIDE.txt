How YOLO Handles Augmentation:

1. Image Transformation
   - Apply rotation/scale/translation/flip to image pixels
   - Use OpenCV/PIL for fast GPU-accelerated transforms

2. Label Transformation (Automatic!)
   - Original: x_center, y_center, width, height (normalized)
   - Apply same transform matrix to box corners
   - Convert back to x_center, y_center, width, height
   - Re-normalize to 0.0-1.0 range
   - Clip to image boundaries

3. Example: 15Â° Rotation
   Before: class_id 0.5 0.5 0.2 0.1
           (box at center, 20% wide, 10% tall)
   
   After rotation:
           class_id 0.52 0.48 0.22 0.12
           (box rotated, slightly different center/size)

4. Validation (During Training)
   - YOLO checks if box is still valid
   - Removes boxes that became too small
   - Removes boxes outside image boundaries
   - Ensures no degenerate boxes
